dms_struct documentation¶

dms_struct is a Python package written by the Bloom lab to view deep mutational scanning and mutational antigenic profiling data on protein structures.

dms_struct is essentially a collection of code that simplifies the use of nglview to visualize selection on protein structures in a Jupyter notebook.

See below for information and examples of how to use this package.
